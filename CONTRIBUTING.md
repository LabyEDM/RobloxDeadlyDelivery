# Contributing

## Quick start (Windows PowerShell)
- Run: .\scripts\serve.ps1
- Open the local URL shown in the terminal.

## Editing pages
- Wiki pages live under docs/
- Keep pages factual, concise, and consistent.
- Prefer tables for lists (enemies, bosses, tools, etc.)
- Add sources/links when possible.

## Style
- Use headings consistently (H2 for major sections, H3 for subsections).
- Use ++W++ style for keys (enabled by pymdownx.keys).
